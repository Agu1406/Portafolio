Creación y uso de tablas en HTML


Las tablas en HTML son una forma de organizar y presentar la información similares a las tablas en Excel, estás se pueden crear utilizando las etiquetas <table> (se debe cerrar), dentro de una tabla podemos crear filas y columnas usando <tr> para las columnas e internamente en ellas usando <th> o <td> para crear las filas dentro de las mismas, ejemplo.

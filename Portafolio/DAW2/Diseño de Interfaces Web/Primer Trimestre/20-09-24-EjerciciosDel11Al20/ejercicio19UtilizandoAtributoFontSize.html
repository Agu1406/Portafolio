Probando el Font Size

Es cierto que los titulos y parrafos tienen tamaños predeterminados en HTML, por ejemplo desde los H1 hasta los H6 hay un cambio de tamaño decreciente, pero ¿Y si por estetica queremos cambiar el tamaño de estos elementos? para ellos utilizamos un atributo style modificando el "font size" con tamaños estaticos, ejemplo, pixeles o tamaños responsive, en este ejemplo el H1 tiene un "font size" de 160 pixeles.

Probando los atributos de "src"

Cuando insertamos imagenes en nuestro sitio web y utilizamos la etiqueta "img" esta posee varios atributos, entre ellos esta el texto alternativo que se muestra cuando la imagén no carga correctamente y el "src" que nos permite indicar que imagén deseamos mostrar con una ruta local en nuestro dispositivo o un enlace directo a la imagén por ejemplo alojada en un servidor o sitio web tal y como vemos a continuación.

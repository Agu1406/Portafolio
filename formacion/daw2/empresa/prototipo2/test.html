Prueba de Servidor Web

✓ Si puedes ver esta página, el servidor web está funcionando correctamente.

Herramientas de diagnóstico

  • Prueba de PHP - Verifica si PHP está funcionando correctamente
  • Prueba de MySQL - Verifica la conexión a MySQL y la base de datos
  • Configuración de la base de datos - Verifica y corrige la configuración

Acceso directo a la aplicación

  • Acceso directo a public/index.php - Intenta acceder directamente al punto de entrada
  • Acceso a través de index.php - Accede a través del archivo de redirección principal

Posibles problemas y soluciones

Si PHP no funciona:

  • Verifica que PHP esté instalado y configurado correctamente.
  • Asegúrate de que el módulo PHP esté habilitado en el servidor web.
  • Reinicia el servidor web.

Si MySQL no funciona:

  • Verifica que el servidor MySQL esté en ejecución.
  • Comprueba que las credenciales (usuario y contraseña) sean correctas.
  • Asegúrate de que la base de datos 'manosalaolla' exista.
  • Verifica que las tablas necesarias estén creadas.

Si la aplicación no carga:

  • Verifica que el módulo mod_rewrite de Apache esté habilitado.
  • Comprueba que los archivos .htaccess tengan los permisos correctos.
  • Asegúrate de que la configuración de la URL base sea correcta.
  • Verifica que todos los archivos necesarios existan y sean legibles.

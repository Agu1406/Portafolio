Ejercicio 54 - Tamaños de Botones

Personalización de Tamaños en Botones

El tamaño de los botones puede ajustarse fácilmente mediante la propiedad font-size de CSS, que afecta tanto al texto como a las proporciones generales del botón.

Botones con Diferentes Tamaños

Observa cómo el tamaño del botón cambia proporcionalmente al tamaño de la fuente:

  • 10px: Tamaño muy pequeño, adecuado para espacios reducidos
  • 12px: Tamaño pequeño, útil para botones secundarios
  • 16px: Tamaño estándar, recomendado para la mayoría de los botones
  • 20px: Tamaño grande, ideal para botones destacados
  • 24px: Tamaño muy grande, para botones principales o llamadas a la acción

Código CSS

.button {
    background-color: #4CAF50;
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    display: inline-block;
    cursor: pointer;
}

.button1 { font-size: 10px; }
.button2 { font-size: 12px; }
.button3 { font-size: 16px; }
.button4 { font-size: 20px; }
.button5 { font-size: 24px; }

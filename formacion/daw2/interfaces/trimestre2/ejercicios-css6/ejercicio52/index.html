Ejercicio 52 - Botones CSS

Estilos de Botones en CSS

Los botones son elementos interactivos fundamentales en cualquier interfaz web. CSS permite estilizarlos de manera consistente independientemente del elemento HTML utilizado.

Botones con Diferentes Elementos HTML

Enlace como Botón

Observa cómo la clase .button aplica el mismo estilo a diferentes elementos HTML:

  • Elemento <button> sin clase (estilo predeterminado del navegador)
  • Elemento <a> con clase .button
  • Elemento <button> con clase .button
  • Elemento <input type="button"> con clase .button

Código CSS

.button {
    background-color: #4CAF50;
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
}

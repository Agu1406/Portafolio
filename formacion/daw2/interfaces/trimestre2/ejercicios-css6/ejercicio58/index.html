Ejercicio 58 - Botones con Sombra

Personalización de Sombras en Botones

La propiedad box-shadow permite añadir sombras a los botones, creando efectos de profundidad y elevación que mejoran la experiencia del usuario.

Botones con Diferentes Efectos de Sombra

Observa los diferentes comportamientos de las sombras:

  • Botón con Sombra: Muestra una sombra permanente que da sensación de elevación
  • Botón con Sombra al Hover: Muestra una sombra solo cuando el cursor pasa por encima, simulando que el botón se eleva al interactuar con él

Múltiples Sombras

La propiedad box-shadow tiene varios componentes:

  • Desplazamiento horizontal: Distancia de la sombra en el eje X (positivo = derecha, negativo = izquierda)
  • Desplazamiento vertical: Distancia de la sombra en el eje Y (positivo = abajo, negativo = arriba)
  • Desenfoque: Radio de desenfoque de la sombra (mayor valor = más difusa)
  • Propagación: Tamaño de la sombra (opcional)
  • Color: Color de la sombra, generalmente con transparencia (rgba)
  • Inset: Palabra clave para crear sombras interiores (opcional)

La sintaxis completa es: box-shadow: h-offset v-offset blur spread color inset;

Código CSS

.button {
    background-color: #4CAF50;
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
    -webkit-transition-duration: 0.4s; 
    transition-duration: 0.4s;
}

.button1 {
    box-shadow: 0 8px 16px 0 rgba(0,0,0,0.2), 0 6px 20px 0 rgba(0,0,0,0.19);
}

.button2:hover {
    box-shadow: 0 12px 16px 0 rgba(0,0,0,0.24), 0 17px 50px 0 rgba(0,0,0,0.19);
}

Ejercicio 60 - Ancho de Botones

Personalización del Ancho de Botones

La propiedad width permite controlar el ancho de los botones, adaptándolos a diferentes necesidades de diseño y dispositivos.

Botones con Diferentes Valores de Ancho

La propiedad width puede tener diferentes valores:

  • Píxeles (250px): Establece un ancho fijo, independiente del tamaño del contenedor
  • Porcentaje (50%): Establece un ancho relativo al elemento padre, útil para diseños responsivos
  • Porcentaje (100%): Ocupa todo el ancho disponible del elemento padre

Cómo Funciona la Propiedad Width

width: 100px
width: 200px
width: 300px
width: 25%
width: 50%
width: 75%

Consideraciones importantes sobre el ancho de los botones:

  • Valores fijos (px): Útiles cuando necesitas un tamaño específico y consistente
  • Valores relativos (%): Ideales para diseños responsivos que se adaptan a diferentes tamaños de pantalla
  • Diseño móvil: En dispositivos móviles, los botones con ancho del 100% son más fáciles de tocar
  • Accesibilidad: Los botones deben tener un tamaño suficiente para ser fácilmente clicables (mínimo 44x44px según WCAG)

Prueba a cambiar el tamaño de la ventana del navegador para ver cómo se comportan los botones con diferentes anchos.

Código CSS

.button {
    background-color: #4CAF50; 
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
}

.button1 {width: 250px;}
.button2 {width: 50%;}
.button3 {width: 100%;}

Ejercicio 55 - Padding en Botones

Personalización del Relleno en Botones

El espacio interno de los botones puede ajustarse mediante la propiedad padding de CSS, permitiendo crear botones con diferentes proporciones y tamaños.

Botones con Diferentes Valores de Padding

La propiedad padding puede tener diferentes valores:

  • 10px 24px: 10px arriba/abajo, 24px izquierda/derecha
  • 12px 28px: 12px arriba/abajo, 28px izquierda/derecha
  • 14px 28px: 14px arriba/abajo, 28px izquierda/derecha
  • 32px 16px: 32px arriba/abajo, 16px izquierda/derecha (botón alto)
  • 16px: 16px en todas las direcciones (botón cuadrado)

Cómo Funciona el Padding

Contenido

El padding es el espacio entre el contenido y el borde del elemento. Cuando se especifican dos valores:

  • El primer valor afecta al padding vertical (arriba y abajo)
  • El segundo valor afecta al padding horizontal (izquierda y derecha)

Con un solo valor, se aplica el mismo padding en todas las direcciones.

Código CSS

.button {
    background-color: #4CAF50;
    border: none;
    color: white;
    text-align: center;
    display: inline-block;
    font-size: 16px;
    cursor: pointer;
}

.button1 { padding: 10px 24px; }
.button2 { padding: 12px 28px; }
.button3 { padding: 14px 28px; }
.button4 { padding: 32px 16px; }
.button5 { padding: 16px; }

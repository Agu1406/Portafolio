Ejercicio 49 - Propiedades de Animación

Propiedades de Animación en CSS

CSS ofrece varias propiedades para controlar el comportamiento de las animaciones, desde su duración hasta cómo se comportan antes y después de ejecutarse.

Animación Original (Propiedades Individuales)

Espera 2s...

animation-name: example;
animation-duration: 5s;
animation-timing-function: linear;
animation-delay: 2s;
animation-iteration-count: infinite;
animation-direction: alternate;

Este es el ejemplo original del ejercicio. La animación utiliza seis propiedades diferentes para controlar su comportamiento.

Notación Abreviada (Shorthand)

Básico
Completo

animation: example 5s linear 2s infinite alternate;

La propiedad animation permite definir todas las propiedades en una sola línea, en el siguiente orden: name, duration, timing-function, delay, iteration-count, direction, fill-mode, play-state.

Propiedad animation-fill-mode

none
forwards
backwards
both

animation-fill-mode: none | forwards | backwards | both;

Determina cómo se aplican los estilos antes y después de la ejecución de la animación.

Propiedad animation-play-state

running
paused
hover me

animation-play-state: running | paused;

Permite pausar y reanudar la animación. El último ejemplo se inicia al pasar el cursor por encima.

Múltiples Animaciones

3 animaciones

animation: move-diagonal 5s infinite, color-change 3s infinite, pulse 2s infinite;

Es posible aplicar varias animaciones a un mismo elemento separándolas con comas. Cada animación puede tener sus propias propiedades.

Tabla de Propiedades de Animación

Propiedad Descripción Valores
animation-name Especifica el nombre de la animación (@keyframes) keyframename | none
animation-duration Especifica la duración de la animación time (s o ms)
animation-timing-function Especifica la curva de velocidad linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier | steps
animation-delay Especifica el retraso antes de iniciar time (s o ms)
animation-iteration-count Especifica el número de repeticiones number | infinite
animation-direction Especifica la dirección de reproducción normal | reverse | alternate | alternate-reverse
animation-fill-mode Especifica los estilos antes/después none | forwards | backwards | both
animation-play-state Especifica si está en pausa o reproducción running | paused

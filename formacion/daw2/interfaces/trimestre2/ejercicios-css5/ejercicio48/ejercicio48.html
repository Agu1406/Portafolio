Ejercicio 48 - Funciones de Temporización

Propiedad animation-timing-function

La propiedad animation-timing-function especifica la curva de velocidad de una animación, determinando cómo se calculan los valores intermedios.

Funciones de Temporización Básicas

linear
ease
ease-in
ease-out
ease-in-out

animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out;

Estos son los ejemplos originales del ejercicio. Cada caja se mueve con una función de temporización diferente.

Funciones Step

step-start
step-end
steps(5)
steps(10)

animation-timing-function: step-start | step-end | steps(n, end);

Las funciones step dividen la animación en segmentos discretos, creando un efecto de "salto" en lugar de un movimiento suave.

Funciones Cubic-Bezier Personalizadas

bounce
elastic
back

animation-timing-function: cubic-bezier(x1, y1, x2, y2);

La función cubic-bezier permite crear curvas de temporización personalizadas definiendo cuatro puntos de control.

Aplicaciones Prácticas

  • linear: Útil para animaciones continuas como rotaciones o movimientos constantes.
  • ease-in: Ideal para elementos que salen de la pantalla o desaparecen.
  • ease-out: Perfecto para elementos que entran en la pantalla o aparecen.
  • ease-in-out: Excelente para transiciones suaves entre estados.
  • steps: Útil para animaciones de sprites o efectos de "fotograma a fotograma".

Compatibilidad

Todas las funciones de temporización básicas tienen buen soporte en navegadores modernos. Para máxima compatibilidad:

  • Usa prefijos de navegador para cubic-bezier en navegadores antiguos.
  • Proporciona alternativas para navegadores que no soporten steps().
  • Prueba tus animaciones en diferentes dispositivos y navegadores.

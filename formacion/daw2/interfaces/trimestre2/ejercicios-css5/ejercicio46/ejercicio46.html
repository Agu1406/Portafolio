Ejercicio 46 - Animaciones CSS

Propiedades de Animación en CSS

CSS permite crear animaciones complejas mediante la definición de keyframes y el control de diversas propiedades de animación.

Animación Básica

animation-name: example;
animation-duration: 4s;
animation-iteration-count: infinite;

Este es el ejemplo original del ejercicio. La animación mueve un cuadrado en un patrón rectangular mientras cambia de color.

Funciones de Temporización (timing-function)

animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out

Controla la aceleración de la animación. De arriba a abajo: linear, ease, ease-in, ease-out, ease-in-out.

Dirección de Animación

animation-direction: normal | reverse | alternate | alternate-reverse

Determina si la animación se reproduce hacia adelante, hacia atrás o alterna entre ambas direcciones.

Fill Mode

animation-fill-mode: none | forwards | backwards | both

Define cómo se aplican los estilos antes y después de la ejecución de la animación. De izquierda a derecha: none, forwards, backwards, both.

Play State

Pasa el cursor por encima para pausar la animación

animation-play-state: running | paused

Permite pausar y reanudar la animación. En este ejemplo, la animación se pausa al pasar el cursor por encima.

Shorthand (Notación Abreviada)

animation: pulsar 2s ease-in-out 0s infinite alternate both running;

La propiedad animation permite definir todas las propiedades de animación en una sola línea, en el siguiente orden: name, duration, timing-function, delay, iteration-count, direction, fill-mode, play-state.

Ejercicio 39 - Transformaciones 3D

Métodos de Transformación 3D en CSS

CSS permite aplicar transformaciones en tres dimensiones a los elementos HTML, creando efectos de profundidad y perspectiva.

Rotación 2D vs 3D

Normal
rotate(20deg)

transform: rotate(20deg);

Este es el ejemplo original del ejercicio. La rotación 2D gira el elemento en el plano de la página.

rotateX()

rotateX(45deg)

transform: rotateX(45deg);

Rota el elemento alrededor del eje X (horizontal), creando un efecto de inclinación hacia adelante o atrás.

rotateY()

rotateY(45deg)

transform: rotateY(45deg);

Rota el elemento alrededor del eje Y (vertical), creando un efecto de giro lateral.

rotateZ()

rotateZ(45deg)

transform: rotateZ(45deg);

Rota el elemento alrededor del eje Z (perpendicular a la pantalla), similar a rotate() en 2D.

translateZ()

translateZ(50px)

transform: translateZ(50px);

Mueve el elemento a lo largo del eje Z, acercándolo o alejándolo del espectador.

translate3d()

translate3d(30px, 20px, 50px)

transform: translate3d(30px, 20px, 50px);

Mueve el elemento en los tres ejes (X, Y, Z) simultáneamente.

perspective()

perspective(500px)

transform: perspective(500px) rotateY(30deg);

Aplica una perspectiva a la transformación, haciendo que las partes más alejadas parezcan más pequeñas.

Pasa el cursor (Frente)
¡Volteada! (Reverso)

transform-style: preserve-3d; backface-visibility: hidden;

Ejemplo de tarjeta que se voltea en 3D al pasar el cursor. Combina varias propiedades 3D.

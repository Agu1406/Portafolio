Ejercicio 29 - Gradientes Avanzados

CSS Conic Gradients & Radial Gradient with Stops

CSS conic gradients create color transitions that rotate around a center point, and radial gradients can use color stops for more control.

Radial Gradient with Stops

radial-gradient(red 5%, yellow 15%, green 60%);

This is the original example from the exercise.

Basic Conic Gradient

conic-gradient(red, yellow, green, blue, purple, red);

Conic Gradient with Angle

conic-gradient(from 45deg, red, yellow, green, blue, purple, red);

Conic Gradient with Position

conic-gradient(at 60% 40%, red, yellow, green, blue, purple, red);

Pie Chart with Conic Gradient

conic-gradient(red 0deg, red 90deg, yellow 90deg, ...)

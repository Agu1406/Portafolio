Ejercicio 27 - Gradientes Multicolor

CSS Linear Gradients with Multiple Colors

CSS linear gradients can include multiple color stops to create more complex color transitions.

Three Color Gradient

linear-gradient(red, yellow, green);

Rainbow Gradient

linear-gradient(red, yellow, green, blue, indigo, violet);

Gradient with Color Stops

linear-gradient(red 10%, green 85%, blue 90%);

Color stops control where each color begins and ends in the gradient.

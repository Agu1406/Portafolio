Ejercicio 28 - Gradiente Radial

CSS Radial Gradients

CSS radial gradients create color transitions that radiate from a central point.

Default Radial Gradient

radial-gradient(red, yellow, green);

Circle Radial Gradient

radial-gradient(circle, red, yellow, green);

Positioned Radial Gradient

radial-gradient(at top left, red, yellow, green);

Sized Radial Gradient

radial-gradient(circle closest-side, red, yellow, green);

Farthest Corner Radial Gradient

radial-gradient(circle farthest-corner at 50% 50%, red, yellow, green);

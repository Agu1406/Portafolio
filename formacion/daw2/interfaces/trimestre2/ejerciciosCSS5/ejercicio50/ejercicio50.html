Tooltips en CSS

Los tooltips son pequeñas ventanas emergentes que aparecen cuando el usuario pasa el cursor sobre un elemento, proporcionando información adicional.

Implementación de Tooltips

Para crear un tooltip básico en CSS:

  1. Crea un contenedor con position: relative
  2. Añade el tooltip como un elemento hijo con position: absolute
  3. Oculta el tooltip con visibility: hidden
  4. Muestra el tooltip al pasar el cursor con :hover
  5. Posiciona el tooltip según sea necesario
  6. Añade una flecha con pseudo-elementos si lo deseas

Consideraciones de Accesibilidad

Al implementar tooltips, considera lo siguiente:

  • Asegúrate de que la información crítica no esté solo en tooltips
  • Proporciona alternativas para usuarios de teclado y lectores de pantalla
  • Considera usar atributos aria-label o aria-describedby
  • Asegúrate de que haya suficiente contraste de color
  • Evita tooltips que desaparezcan demasiado rápido

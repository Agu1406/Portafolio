Ejercicio 25 - Elliptical Borders

The border-radius Property with Elliptical Shapes

The border-radius property can create elliptical corners by specifying different horizontal and vertical radii.

Horizontal Elliptical Border

border-radius: 50px / 15px;

Vertical Elliptical Border

border-radius: 15px / 50px;

Circle/Ellipse Border

border-radius: 50%;

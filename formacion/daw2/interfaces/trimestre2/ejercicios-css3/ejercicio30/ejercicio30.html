Ejercicio 30 - Gradientes Repetitivos

CSS Repeating Gradients

CSS repeating gradients create patterns by repeating color stops indefinitely.

Repeating Radial Gradient

repeating-radial-gradient(red, yellow 10%, green 15%);

This is the original example from the exercise.

Repeating Linear Gradient (45deg)

repeating-linear-gradient(45deg, red, yellow 10%, green 15%);

Repeating Linear Gradient (Horizontal)

repeating-linear-gradient(to right, red, yellow 10%, green 15%);

Repeating Conic Gradient

repeating-conic-gradient(red, yellow 10%, green 15%);

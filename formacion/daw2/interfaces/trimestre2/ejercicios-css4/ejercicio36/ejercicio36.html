Ejercicio 36 - Ajuste de Texto

Propiedades de Ajuste de Texto en CSS

CSS ofrece varias propiedades para controlar cómo se ajusta el texto dentro de sus contenedores.

word-wrap: break-word

Este párrafo contiene una palabra muy larga: thisisaveryveryveryveryveryveryverylongword. La palabra larga se romperá y se ajustará a la siguiente línea.

word-wrap: break-word;

Este es el ejemplo original del ejercicio. Permite que las palabras largas se rompan y continúen en la siguiente línea.

word-break: break-all

Este párrafo contiene una palabra muy larga: thisisaveryveryveryveryveryveryverylongword. La palabra se romperá en cualquier carácter.

word-break: break-all;

Rompe las palabras en cualquier carácter, no solo entre palabras.

overflow-wrap: break-word

Este párrafo contiene una palabra muy larga: thisisaveryveryveryveryveryveryverylongword. Similar a word-wrap.

overflow-wrap: break-word;

Es el nombre más moderno para word-wrap, con el mismo comportamiento.

hyphens: auto

Este párrafo demuestra la propiedad hyphens que permite la separación silábica automática en palabras largas como supercalifragilisticoespialidoso.

hyphens: auto;

Permite la separación silábica automática con guiones cuando sea necesario.

white-space: nowrap + text-overflow

Este texto no se ajustará a la siguiente línea y se truncará con puntos suspensivos si es demasiado largo.

white-space: nowrap; overflow: hidden; text-overflow: ellipsis;

Evita que el texto se ajuste y muestra puntos suspensivos cuando se desborda.

white-space: pre

Este texto respeta los espacios y saltos de línea tal como están escritos.

white-space: pre;

Respeta los espacios en blanco y saltos de línea tal como están en el código.

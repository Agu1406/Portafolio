Ejercicio 32 - Contorno y Sombra

Efectos de Contorno y Sombra en CSS

CSS permite aplicar efectos de contorno a textos y sombras a elementos para crear profundidad y énfasis visual.

Contorno de Texto

Contorno de texto

-webkit-text-stroke: 1px black;

Este es el ejemplo original del ejercicio.

Contorno de Texto Grueso

Contorno grueso

-webkit-text-stroke: 2px #0066cc;

Contorno de Texto con Color

Contorno color

color: #ffcc00; -webkit-text-stroke: 1px #cc0000;

Sombra de Caja

box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.5);

Este es el ejemplo original del ejercicio.

Sombra Interna

box-shadow: inset 5px 5px 15px rgba(0, 0, 0, 0.3);

Sombra Múltiple

box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3), -5px -5px 15px rgba(0, 0, 255, 0.2);

Sombra de Texto

Sombra de texto

text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);

Ejercicio 33 - Transformaciones 2D

Transformaciones 2D en CSS

CSS permite aplicar transformaciones 2D a elementos para modificar su posición, tamaño, rotación y más.

Pasa el cursor sobre cada caja para ver el efecto inverso o amplificado.

Rotación (rotate)

rotate

transform: rotate(45deg);

Rota el elemento 45 grados en sentido horario.

Escala (scale)

scale

transform: scale(0.8);

Reduce el tamaño del elemento al 80% de su tamaño original.

Traslación (translate)

translate

transform: translate(20px, 20px);

Mueve el elemento 20px a la derecha y 20px hacia abajo.

Sesgo (skew)

skew

transform: skew(10deg, 5deg);

Inclina el elemento 10 grados en el eje X y 5 grados en el eje Y.

Transformaciones Múltiples

múltiple

transform: rotate(15deg) scale(0.9) translate(10px, 10px);

Combina rotación, escala y traslación en una sola transformación.

Origen de Transformación

origen

transform: rotate(45deg); transform-origin: top left;

Rota el elemento desde la esquina superior izquierda en lugar del centro.

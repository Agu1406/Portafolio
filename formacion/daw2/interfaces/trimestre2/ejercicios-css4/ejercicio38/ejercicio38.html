Ejercicio 38 - Transformaciones 2D

Métodos de Transformación 2D en CSS

CSS ofrece varias funciones de transformación que permiten modificar la posición, tamaño y forma de los elementos.

Las cajas sombreadas muestran la posición original antes de la transformación.

translate()

translate(50px, 30px)

transform: translate(50px, 30px);

Mueve el elemento 50px a la derecha y 30px hacia abajo desde su posición original. Este es el ejemplo original del ejercicio.

rotate()

rotate(20deg)

transform: rotate(20deg);

Rota el elemento 20 grados en sentido horario alrededor de su centro.

scale()

scale(0.8, 1.2)

transform: scale(0.8, 1.2);

Escala el elemento al 80% de su ancho original y al 120% de su altura original.

skew()

skew(10deg, 5deg)

transform: skew(10deg, 5deg);

Inclina el elemento 10 grados en el eje X y 5 grados en el eje Y.

transform-origin

transform-origin

transform: rotate(45deg); transform-origin: top left;

Cambia el punto de origen de la transformación a la esquina superior izquierda.

matrix()

matrix()

transform: matrix(1, 0.2, 0.2, 1, 20, 20);

Combina todas las transformaciones 2D en una sola usando una matriz de transformación.

Transformaciones Múltiples

Múltiples transformaciones

transform: translate(30px, 20px) rotate(15deg) scale(0.9);

Aplica varias transformaciones en secuencia (primero traslación, luego rotación y finalmente escala).

Efecto Hover

Pasa el cursor por encima

transform: scale(1.1) rotate(5deg); (al pasar el cursor)

Las transformaciones son excelentes para crear efectos interactivos. Pasa el cursor sobre la caja para ver el efecto.

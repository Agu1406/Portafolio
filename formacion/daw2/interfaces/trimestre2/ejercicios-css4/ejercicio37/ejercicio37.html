Ejercicio 37 - Modos de Escritura

Propiedad writing-mode en CSS

La propiedad writing-mode en CSS permite cambiar la dirección y orientación del texto, lo que es útil para idiomas con diferentes sistemas de escritura.

Horizontal (Predeterminado)

Este es un texto con el modo de escritura predeterminado (horizontal-tb). El texto fluye de izquierda a derecha y de arriba hacia abajo.

writing-mode: horizontal-tb;

Este es el modo de escritura predeterminado para idiomas occidentales.

Vertical de Derecha a Izquierda

Este texto tiene un modo de escritura vertical que fluye de derecha a izquierda. Común en idiomas asiáticos como el japonés tradicional.

writing-mode: vertical-rl;

Este es el ejemplo original del ejercicio.

Vertical de Izquierda a Derecha

Este texto tiene un modo de escritura vertical que fluye de izquierda a derecha. Usado en algunos contextos de diseño moderno.

writing-mode: vertical-lr;

Texto Combinado

Este es un texto normal con un elemento vertical integrado en el flujo del texto horizontal. Esto permite combinar diferentes modos de escritura en un mismo párrafo.

span { writing-mode: vertical-rl; display: inline-block; }

Combina texto horizontal con elementos verticales.

Aplicaciones Prácticas

Los diferentes modos de escritura son útiles para:

  • Diseño de sitios web multilingües
  • Diseños creativos y artísticos
  • Títulos y encabezados estilizados
  • Adaptación a diferentes culturas y sistemas de escritura

Compatibilidad

Algunas consideraciones importantes:

  • No todos los navegadores soportan todos los valores
  • sideways-rl y sideways-lr tienen soporte limitado
  • Considerar la legibilidad al usar modos no estándar
  • Probar en diferentes dispositivos y navegadores

Ejercicio 40 - Transiciones y Animaciones

Efectos de Transición y Animación en CSS

CSS permite crear efectos dinámicos mediante transiciones suaves entre estados y animaciones complejas.

Transformación Scale

scale(2, 1.5)

transform: scale(2, 1.5);

Este es el ejemplo original del ejercicio. La transformación scale aumenta el tamaño del elemento.

Transición Simple

Pasa el cursor

transition: transform 0.5s ease;

Al pasar el cursor, el elemento se agranda suavemente gracias a la transición.

Transición Múltiple

Pasa el cursor

transition: transform 0.5s, background-color 0.5s, border-radius 0.5s;

Transición que afecta a varias propiedades simultáneamente.

Transición con Retraso

Pasa el cursor

transition: transform 0.5s ease 0.2s;

La transformación comienza después de un pequeño retraso de 0.2 segundos.

Animación de Rebote

Rebote

animation: bounce 2s infinite ease-in-out;

Animación que hace que el elemento rebote continuamente.

Animación de Rotación

Rotación

animation: rotate 3s infinite linear;

Animación que hace que el elemento gire continuamente.

Animación de Pulso

Pulso

animation: pulse 2s infinite ease-in-out;

Animación que hace que el elemento pulse cambiando de tamaño y color.

Animación Pausada

Pasa el cursor

animation-play-state: paused;

Animación que está pausada hasta que el usuario pasa el cursor por encima.

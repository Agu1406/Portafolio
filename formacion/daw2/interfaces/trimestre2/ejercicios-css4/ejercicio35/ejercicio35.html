Ejercicio 35 - Efectos de Texto

Efectos Avanzados de Texto en CSS

CSS permite aplicar efectos visuales avanzados a textos utilizando gradientes, recortes y sombras.

Texto Arcoíris

Efectos de texto

background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
background-clip: text;
-webkit-background-clip: text;
color: transparent;

Este es el ejemplo original del ejercicio.

Texto Metálico

Efecto metálico

background-clip: text;
-webkit-background-clip: text;
color: transparent;

Texto Recortado

Efecto recortado

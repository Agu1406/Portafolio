Ejercicio 56 - Botones Redondeados

Personalización de Esquinas en Botones

La propiedad border-radius permite añadir esquinas redondeadas a los botones, mejorando su apariencia visual y suavizando su diseño.

Botones con Diferentes Valores de Border-Radius

La propiedad border-radius puede tener diferentes valores:

  • 2px: Esquinas ligeramente redondeadas, casi imperceptibles
  • 4px: Esquinas suavemente redondeadas, sutiles
  • 8px: Esquinas moderadamente redondeadas, visibles
  • 12px: Esquinas muy redondeadas, pronunciadas
  • 50%: Esquinas completamente redondeadas, creando un botón circular u ovalado

La propiedad border-radius define el radio de las esquinas del elemento:

  • Valores pequeños crean esquinas ligeramente redondeadas
  • Valores más grandes crean esquinas más redondeadas
  • Un valor de 50% en un elemento cuadrado crea un círculo perfecto
  • En elementos rectangulares, 50% crea una forma ovalada

Código CSS

.button {
    background-color: #4CAF50; 
    border: none;
    color: white;
    padding: 20px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
}

.button1 {border-radius: 2px;}
.button2 {border-radius: 4px;}
.button3 {border-radius: 8px;}
.button4 {border-radius: 12px;}
.button5 {border-radius: 50%;}

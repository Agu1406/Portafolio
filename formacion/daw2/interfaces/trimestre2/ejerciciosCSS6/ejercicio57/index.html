Ejercicio 57 - Botones con Bordes

Personalización de Bordes en Botones

La propiedad border permite añadir bordes de diferentes colores a los botones, creando un estilo distintivo y elegante.

Botones con Diferentes Colores de Borde

La propiedad border puede tener diferentes colores:

  • Verde (#4CAF50): Representa acciones positivas o de confirmación
  • Azul (#008CBA): Representa acciones informativas o de navegación
  • Rojo (#f44336): Representa acciones de alerta o eliminación
  • Gris (#e7e7e7): Representa acciones secundarias o deshabilitadas
  • Negro (#555555): Representa acciones neutras o alternativas

Cómo Funciona la Propiedad Border

2px solid #4CAF50
2px dashed #008CBA
2px dotted #f44336
4px double #555555

La propiedad border tiene tres componentes principales:

  • Ancho: Define el grosor del borde (ej. 2px, 3px, etc.)
  • Estilo: Define el tipo de línea (solid, dashed, dotted, double, etc.)
  • Color: Define el color del borde usando valores hexadecimales, RGB, etc.

La sintaxis completa es: border: ancho estilo color;

Código CSS

.button {
    background-color: #4CAF50;
    border: none;
    color: white;
    padding: 15px 32px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    margin: 4px 2px;
    cursor: pointer;
}

    border: 2px solid #4CAF50;

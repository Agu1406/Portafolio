Ejercicio 51 - Tooltip con Flecha

Tooltip Superior con Flecha Inferior

Pasa el cursor sobre el texto para ver el tooltip:

Pasa el cursor sobre mí Texto del tooltip

Explicación

Este tooltip aparece encima del elemento y tiene una flecha que apunta hacia abajo. Se implementa usando:

  • Posicionamiento absoluto para el tooltip
  • Un pseudo-elemento ::after para crear la flecha
  • Bordes transparentes y coloreados para formar el triángulo de la flecha
  • La propiedad visibility para mostrar/ocultar el tooltip
